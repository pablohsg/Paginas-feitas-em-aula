Pseudo classes

Sintaxe

            
            seletor:pseudo-classe{
                propriedade:valor;
            }

Pseudo classe :hover

Acontece quando passo o mouse por cima de um elemento.

Pseudo classe :active

Acontece quando eu clico sobre o elemento.

Pseudo classe :focus

Acontece quando eu quero dar foco em um elemento.

Pseudo classe :link e :visited

Utilizamos para definir a estilização de um link.

link para quando um link não foi visitado.

visited para quando um link foi visitado.
